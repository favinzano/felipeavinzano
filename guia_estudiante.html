01 · Claude — Anthropic

claude.ai · Proyectos · Estilos · Pensamiento Extendido · ~40 min

0/7
▼

Acceso

claude.ai (web) · App iOS / Android. Necesitas número de teléfono para SMS de verificación.

Planes

Free (gratuito, suficiente para hoy) · Pro $20/mes · API pago por tokens.

Configuración paso a paso
01
Crear cuenta y verificar por SMS
Ir a claude.ai → "Crear cuenta" → seleccionar país de residencia → ingresar número de teléfono → ingresar código de verificación recibido por SMS.
02
Completar tu perfil
Esquina inferior izquierda → clic en tu nombre → Configuración → agregar nombre completo, cargo actual, seleccionar tema visual (claro u oscuro).
03
Preferencias de perfil GLOBAL
Configuración → "¿Qué preferencias debe considerar Claude?" → Escribe: tu industria, tono que prefieres (directo / formal / académico), que siempre cite fuentes de autoridad, y los términos técnicos específicos de tu sector. Esta configuración aplica a TODOS tus chats automáticamente.
04
Crear tu primer Estilo
Caja de texto del chat → ícono "+" → "Crear y editar estilos" → nombra el estilo (ej: "Profesional formal") → describe: tono, tipo de audiencia, formato esperado de las respuestas. Puedes crear múltiples estilos para contextos distintos.
05
Crear tu primer Proyecto CLAVE
Panel lateral izquierdo → "Proyectos" → "Nuevo proyecto" → define un nombre → agrega instrucciones específicas (system prompt) → sube archivos propios (PDF, Word, Excel) como base de conocimiento. Sin Proyectos, cada chat empieza de cero. Con Proyectos, continúa donde lo dejaste.
06
Vincular conectores
Sección "Conectores" en la interfaz → vincular según lo que uses: Google Drive, Gmail, Calendar, GitHub, Notion, Canva, Slack, HubSpot. Los add-ons de Excel y PowerPoint requieren plan Pro.
07
Pensamiento Extendido PLAN PRO
En el chat → ícono de reloj en la caja de texto. Claude dedica tiempo adicional de razonamiento profundo antes de responder. Vista dividida: proceso documentado (izquierda) + informe final exportable (derecha). Combinar con un Estilo formal = resultado de máxima calidad.
⚠️

Crítico — Configura esto antes de cualquier otra cosa

  • Configuración → Privacidad → DESACTIVAR "Ayudar a mejorar Claude". Por defecto está ACTIVADO y tus prompts y documentos alimentan el entrenamiento del modelo.
  • Modo Incógnito: ícono de fantasma en la barra superior → consultas que no quedan guardadas en historial ni memoria.
  • Claude Cowork (agente local): crear carpeta "sandbox" aislada con permisos limitados solo a esa carpeta.
Pros y contras validados

Pros

+
Excelente para documentos largos y análisis de alto contexto.
+
Alta precisión en temas técnicos y codificación.
+
Ventana de contexto de 200k–1M tokens según el modelo.

Contras

−
Sin generación nativa de imagen o audio.
−
Puede ser cauteloso en el tono en algunos contextos.
−
Puede ser más lento para ideación rápida que ChatGPT.
Casos de uso concretos
  • Analizar contrato de 90 páginas y señalar los riesgos.
  • Convertir investigación densa en publicaciones atractivas.
  • Refactorizar base de código grande con contexto completo.
Movimiento Maestro
Usar Claude para revisar y refactorizar código complejo con conciencia total del contexto.

02 · ChatGPT — OpenAI

chatgpt.com · Instrucciones personalizadas · Memoria · 2FA · ~35 min

0/7
▼

Acceso

chatgpt.com o app "ChatGPT" de OpenAI (Play Store / App Store). Registrar con email o Google.

Planes

Free (GPT-5 con límites; cae a GPT-5 mini al agotarlos) [VERIFICADO] · Plus $20/mes (GPT-5 Thinking + imágenes) · Business.

Configuración paso a paso
01
Crear cuenta y configurar idioma
chatgpt.com → "Sign Up" → registrar con email o "Continuar con Google" → completar nombre y fecha de nacimiento. Luego: Perfil → Configuración → General → cambiar idioma a Español y seleccionar tema visual.
02
Instrucciones personalizadas — Cuadro 1 GLOBAL
Perfil → "Personalizar ChatGPT" → Cuadro 1 (¿Qué quieres que ChatGPT sepa sobre ti?): escribe tu nombre, industria, ubicación, cargo actual y objetivos de uso de la IA. Este cuadro define el contexto de quién pregunta.
03
Instrucciones personalizadas — Cuadro 2 GLOBAL
Cuadro 2 (¿Cómo quieres que ChatGPT responda?): escribe el tono que prefieres (directo, profesional, sin rodeos), que exija siempre citar fuentes, que no invente datos que no tenga, y que limite el uso de emojis. Este cuadro define cómo debe responderte.
04
Activar Memoria CLAVE
En el menú de personalización → activar "Hacer referencia a las memorias guardadas". ChatGPT recordará contexto, tus preferencias y datos importantes entre conversaciones. También puedes editar o eliminar memorias específicas desde ese mismo menú.
05
Configurar voz y conectores
Configuración → Habla / Voz → confirmar idioma → seleccionar voz sintética preferida (Ember, Cove, etc.). Luego: "Explorar ChatGPT" o "Aplicaciones conectadas" → vincular Google Drive, Canva u otras apps externas que uses.
06
Autenticación de dos factores — 2FA OBLIGATORIO
Configuración → Seguridad → Autenticación multifactor → activar → copiar la clave de configuración → abrir Google Authenticator → agregar cuenta → pegar clave → obtener código de 6 dígitos → ingresar en ChatGPT para verificar. Guardar el código de recuperación en un lugar seguro fuera del dispositivo. Sin él, si pierdes el teléfono, pierdes el acceso.
07
Modo de razonamiento avanzado PLAN PLUS
En el selector de modelo (parte superior del chat) → GPT-5 Thinking. Equivalente al Pensamiento Extendido de Claude: el modelo razona en profundidad antes de responder. Ideal para análisis complejos, decisiones estratégicas y código con múltiples dependencias.
⚠️

Crítico — Bloquear entrenamiento de datos

  • Configuración → Controles de datos → DESACTIVAR "Mejorar el modelo para todos".
  • En algunas versiones de la interfaz aparece como "Historial de chat y formación". En cualquier variación: deshabilitar.
  • Esto protege tus conversaciones y archivos de alimentar versiones futuras del modelo.
Pros y contras validados

Pros

+
Rápido, intuitivo, multimodal: texto, voz, imágenes.
+
Amplio ecosistema de GPTs personalizados.
+
Memoria robusta y herramientas de navegación integradas.

Contras

−
Aún puede alucinar o perder contexto. [VERIFICADO]
−
Se requiere revisión humana para datos críticos. [VERIFICADO]
−
Plan gratuito da acceso a GPT-5 con límites; cae a GPT-5 mini al agotarlos. [VERIFICADO]
Casos de uso concretos
  • Convertir notas en bruto en guía de incorporación.
  • Resumir sitio web y extraer historia de ventas convincente.
  • Conectar con Zapier + GPTs para autogenerar correos de alcance.
Movimiento Maestro
Crear un GPT Personalizado para tu negocio con instrucciones y base de conocimiento propia.

03 · Gemini — Google

gemini.google.com · Instrucciones · Extensiones · Privacidad · ~25 min

0/6
▼

Acceso

gemini.google.com (web) · App "Google Gemini" (Play Store / App Store) · App escritorio (Chrome) · Asistente de voz ("Ey Google").

Modelos

Flash: rápido, gratuito ilimitado. Pro / Razonamiento: análisis complejos y tareas exigentes. Seleccionar en esquina superior izquierda del chat.

Configuración paso a paso
01
Acceso y configuración de idioma
gemini.google.com → iniciar sesión con tu cuenta Google (Gmail o Workspace) → aceptar términos. Foto de perfil → Configuración → Idiomas → seleccionar Español. Ajustar tema claro u oscuro.
02
Seleccionar modelo según tarea CLAVE
Esquina superior izquierda del chat → menú desplegable → Flash para consultas rápidas y uso diario (gratuito ilimitado), Pro o Razonamiento para análisis complejos, documentos extensos y tareas que requieren razonamiento profundo.
03
Instrucciones para Gemini GLOBAL
Menú lateral → "Instrucciones para Gemini" → escribe: tu ocupación actual, formato de respuesta preferido (corto, directo, sin relleno), tono y cualquier contexto relevante. Se aplican automáticamente a todos los chats. Equivalente a las Instrucciones Personalizadas de ChatGPT.
04
Activar extensiones de Google Workspace
Configuración y ayuda → "Aplicaciones conectadas" → activar las que usas: Google Drive, Gmail, Docs, Calendar, YouTube, Google Maps, Vuelos, Hoteles. Una vez activadas, usa @NombreServicio dentro del chat para invocar una extensión específica. Gemini es la única IA que ejecuta acciones dentro de tu ecosistema Google sin copiar y pegar.
05
Instalar app de escritorio (opcional)
Chrome → menú de tres puntos (⋮) → "Guardar y compartir" → "Instalar página como aplicación" → Instalar. Gemini queda como app de escritorio accesible desde la barra de tareas. Para activar "Ey Google": App Google → foto de perfil → Configuración → Asistente de Google → seleccionar Gemini.
06
Chat temporal para consultas sensibles
Ícono de rectángulo punteado en la interfaz del chat → activa el modo de chat temporal. Las consultas realizadas en este modo no se guardan en el historial, no alimentan la memoria y no quedan registradas en tu actividad de Google.
⚠️

Crítico — Controlar historial de actividad

  • Ruta 1: Configuración y ayuda → Actividad → "Desactivar y borrar actividad".
  • Ruta 2: myactivity.google.com/product/gemini para control granular con filtros por fecha.
  • Ajustar período de eliminación automática: 3, 18 o 36 meses según tu preferencia.
  • Regla de oro: NUNCA ingresar contraseñas, datos bancarios ni información personal sensible en ninguna plataforma de IA.
Pros y contras validados

Pros

+
Integración estrecha con Gmail, Docs, Drive y Calendar.
+
Fuerte manejo multimodal: texto, imágenes, audio y video.
+
Excelente para colaboración en equipo y preparación asíncrona.

Contras

−
Funciones avanzadas bloqueadas tras niveles superiores de pago. [VERIFICADO]
−
Aún puliendo la UX del chat general.
−
Más potente dentro del ecosistema Google que fuera. [VERIFICADO]
Casos de uso concretos
  • Resumir hilos de correo largos y adjuntos desde Gmail.
  • Redactar documentos usando el contexto de imágenes o gráficos.
  • Ejecutar flujos de trabajo de investigación multimodales.
Movimiento Maestro
Construir agentes que combinen correo + calendario en un solo prompt usando extensiones de Google Workspace.
